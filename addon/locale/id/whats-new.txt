Apa yang Baru?

Versi 1.3.0
- Menambahkan kompatibilitas penuh untuk NVDA 2026.1 (Python 64-bit).
- Memfaktorkan ulang semua pintasan global menjadi Lapisan Perintah terpadu (Ctrl+Alt+C) untuk mencegah bentrokan. Tekan F1 setelah pemicu lapisan perintah untuk melihat semua perintah yang tersedia.
- Memperkenalkan pengelola Riwayat Papan Klip tak terbatas dengan kemampuan CRUD penuh (H di lapisan perintah).
- Menambahkan Mode Tambah (A di lapisan perintah) untuk mengumpulkan beberapa teks secara mulus dengan menggabungkan salinan baru.
- Merestrukturisasi dialog Editor dengan memindahkan alat ke dalam menu Alat khusus.
- Mengembalikan kemampuan untuk membacakan konten papan klip tanpa batas karakter (S di lapisan perintah).
- Menambahkan pengaturan untuk mengonfigurasi ukuran maksimum Riwayat Papan Klip dan menimpa bahasa tampilan addon terlepas dari NVDA.
- Memperbaiki bug di mana dialog Editor akan macet atau hang setelah berhasil menyimpan file melalui 'Simpan Sebagai...'.
- Mengintegrasikan dialog Dukungan Pengembangan (donasi PayPal) ke dalam alur instalasi.
- Menambahkan penghitungan paragraf pada fitur Informasi (karakter, kata, baris, dan paragraf).
