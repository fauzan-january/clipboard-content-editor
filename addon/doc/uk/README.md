# Редактор вмісту буфера обміну

Автор: Fauzan January

[Прочитати англійською](../en/readme.md)

Цей додаток для NVDA дозволяє редагувати текст у буфері обміну у простому діалозі перед вставкою. Він також працює як потужний менеджер буфера обміну з необмеженою історією, режимом додавання (append mode) та швидкими діями для перегляду інформації, озвучення тексту та його форматування.

## Особливості

- **Редактор буфера обміну**: Редагуйте поточний текст буфера обміну в багаторядковому редакторі за допомогою інструментів Пошук/Заміна, Зміна регістру та Очищення тексту.
- **Історія буфера обміну**: автоматично зберігає скопійовані елементи в списку історії без обмежень на кількість записів із підтримкою операцій CRUD (створення, читання, оновлення, видалення).
- **Режим додавання (Append Mode)**: Легко збирайте кілька текстових фрагментів, автоматично об'єднуючи нові копії з існуючим буфером обміну.
- **Інформація та Озвучення**: Переглядайте детальну інформацію (символи, слова, рядки) та озвучуйте весь текст буфера обміну без обмеження кількості символів.
- **Зберегти у файл**: Зберігайте вміст буфера обміну безпосередньо у файл `.txt`.
- **Командний шар (Command Layer)**: Уніфікована система глобальних гарячих клавіш для запобігання конфліктам клавіатури.

## Як використовувати

1. Натисніть `Ctrl+Alt+C`, щоб активувати командний шар додатка.
2. Натисніть `E`, щоб відкрити редактор.
3. Відредагуйте текст, використовуйте меню `Інструменти` (Tools) для зміни регістру або очищення тексту, або використовуйте Пошук/Заміна.
4. Натисніть «Зберегти» (Ctrl+S), щоб оновити буфер обміну відредагованим вмістом.

## Гарячі клавіші

### Глобальний командний шар
Щоб скористатися цими командами, спочатку натисніть `Ctrl+Alt+C`, а потім одну з наступних клавіш:

- `F1`: Відкрити список команд або повну документацію.
- `A`: Увімкнути/вимкнути режим додавання.
- `E` - Відкрити редактор буфера обміну.
- `R` - Відновити попередній вміст буфера обміну (з резервної копії редактора).
- `I` - Показати інформацію про текст у буфері обміну.
- `S` - Озвучити весь вміст буфера обміну.
- `H` - Відкрити менеджер історії буфера обміну.
- `A` - Увімкнути або вимкнути режим додавання (Append Mode).
- `F1` - Показати список усіх доступних команд.

## Системні вимоги

- NVDA 2025.3 або новішої версії (включно з повною підтримкою NVDA 2026.1 і 64-бітного Python).
- Windows 10 або новіша версія.

### Гарячі клавіші редактора
(Ці клавіші працюють лише тоді, коли відкритий діалог редактора)

- `Alt+T` - Відкрити меню Інструменти (Зміна регістру, Очищення тексту).
- `Alt+I` - Показати інформацію про текст редактора.
- `Alt+F` - Знайти текст.
- `Alt+R` - Замінити текст.
- `Ctrl+S` - Зберегти зміни (кнопка «Зберегти»).
- `Ctrl+Shift+S` - Зберегти вміст як файл (кнопка «Зберегти як»).
- `Esc` - Скасувати (кнопка «Скасувати»).

### Гарячі клавіші менеджера історії
(Ці клавіші працюють лише тоді, коли відкритий діалог історії)

- `Enter` - Відновити вибраний елемент у буфер обміну.
- `Alt+E` - Відкрити вибраний елемент у редакторі.
- `Alt+D` - Видалити вибраний елемент з історії.
- `Alt+C` - Очистити всю історію.

## Налаштування

Відкрийте налаштування NVDA та виберіть відповідну категорію "Додатка":

- Мова додатка (дозволяє використовувати для додатка іншу мову, ніж у NVDA; для цього потрібно перезапустити NVDA).
- Увімкнути звук (Початково: увімкнено).
- Зберігати активність гарячих клавіш, коли кнопки приховані в редакторі (Початково: увімкнено).
- Увімкнути режим захисту (резервне копіювання буфера обміну в редакторі).
- Кількість рівнів резервного копіювання.

## Примітки

- **Режим додавання**: Коли ввімкнено, кожне натискання `Ctrl+C` додаватиме новий текст до нижньої частини вашого поточного буфера обміну замість його заміни. Не забудьте вимкнути його після завершення збору тексту!
- **Менеджер історії**: Історія автоматично відстежує весь текст, який ви копіюєте у своїй системі.
