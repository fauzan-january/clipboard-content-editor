# Редактор вмісту буфера обміну

Автор: Fauzan January

Цей додаток для NVDA дозволяє редагувати текст у буфері обміну у простому діалозі перед вставкою. Також він надає швидкі дії для перегляду інформації, пошуку та заміни тексту.

## Що нового?


— До редактора додано функцію **Зберегти як** (Ctrl+Shift+S), що дозволяє зберігати вміст у форматі .txt або інших типів файлів.
- Додано налаштування для ввімкнення або вимкнення звуків додатка.
- Додано повний звуковий супровід для функцій "Інформація", "Відновлення з резервної копії" та "Замінити все" для забезпечення послідовних звукових сигналів.
— Виправлено посилання «Прочитати [мовою]» в документації, щоб правильно відкривати HTML-файли замість показу помилок «Файл не знайдено» або відкриття необроблених вихідних файлів.
— Кнопки редактора тепер строго впорядковані для кращої навігації: Інформація (Alt+I) -> Знайти -> Замінити -> Зберегти -> Зберегти як -> Скасувати.

## особливості

- Редагувати поточний текст буфера обміну в багаторядковому редакторі.
- Пошук тексту за допомогою спеціального діалогу пошуку.
- Заміна тексту в редакторі.
- Показати інформацію (символи, слова, рядки) про текст із буфера обміну.
- Зберегти вміст буфера обміну у файл.
— Додаткове резервне копіювання буфера обміну (режим захисту) із відновленням.
- Налаштування звуку.
- Глобальні гарячі клавіші для інформації.

## Як використовувати

1. Натисніть `NVDA+E`, щоб відкрити редактор.
2. Відредагуйте текст за потреби.
3. Використовуйте кнопки або гарячі клавіші «Інформація», «Знайти», «Замінити», «Зберегти» (Ctrl+S) або «Скасувати» (Esc).
4. Натисніть «Зберегти» (Ctrl+S), щоб оновити буфер обміну відредагованим вмістом.

## Гарячі клавіші

Глобальні гарячі клавіші (працюють у будь-якому місці NVDA):

- `NVDA+E` - Відкрити редактор буфера обміну.
- `NVDA+I` - Показати інформацію про текст у буфері обміну.
- `NVDA+Z` - Відновити попередній вміст буфера обміну (режим захисту).

Гарячі клавіші редактора:
- `Alt+I` - Показати інформацію про текст редактора.
- `Alt+F` - Знайти текст.
- `Alt+R` - Замінити текст.
- `Ctrl+S` - Зберегти зміни (кнопка «Зберегти»).
- `Ctrl+Shift+S` - Зберегти вміст як файл (кнопка «Зберегти як»).
- `Esc` - Скасувати (кнопка «Скасувати»).

Якщо буфер обміну порожній, інформація оголосить «буфер обміну порожній».

## Налаштування

Відкрийте налаштування NVDA та виберіть відповідну категорію "Додатка":

- Увімкнути звук (Початково: увімкнено).
- Зберігати активність гарячих клавіш, коли кнопки приховані в редакторі (Початково: увімкнено).
- Увімкнути кнопку "Інформація" в редакторі.
- Увімкнути кнопку "Знайти" в редакторі.
- Увімкнути кнопку "Замінити" в редакторі.
- Увімкнути режим захисту (резервне копіювання буфера обміну).
- Кількість рівнів резервного копіювання.

## Режим захисту (резервне копіювання буфера обміну)

Коли режим захисту ввімкнено, додаток зберігає історію вмісту буфера обміну перед збереженням. Ви можете відновити попередній вміст буфера обміну за допомогою `NVDA+Z`. Кількість збережених резервних копій контролюється «Кількістю рівнів резервного копіювання».

## Примітки

- Функція «Знайти та замінити» підтримує регістр символів і працює тільки в тексті редактора.
- У діалозі пошуку введіть текст і натисніть Esc, щоб перейти до наступного збігу.
